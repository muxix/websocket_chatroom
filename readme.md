## 项目简介
  - 本项目是基于 WebSocket 的在线聊天室：
    - 实现服务器和客户端全双工通信，达到实时通讯的目的；
    - 使用 Shell 脚本实现一键部署。
    - 实现在线聊天室基础功能，包括：用户自由进出、同一房间消息实时同步、房间切换。

## 项目部署

```

```
